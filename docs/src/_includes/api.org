#+BEGIN_SRC sh :results output code :exports both

*** Search for a star

Searching is also available to get stars that match a query. Here is an example:

#+BEGIN_SRC sh :results output code :exports both
curl http://star-api.herokuapp.com/api/v1/search?q="Sun" | python -m json.tool
#+END_SRC
